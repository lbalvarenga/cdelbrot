## A command-line ncurses Mandelbrot Set visualizer written in C. 


#### Compilation

In a Linux shell, clone the repository with

```bash
$: git clone https://github.com/lukeathedev/cdelbrot.git
```

then

```bash
$: make run
```

or

```bash
$: make
$: ./mandebrot
```

to compile and run the program.
